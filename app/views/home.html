This site is written with AngularJS, a superb MVW (Model-View-Whatever) javascript framework. It originated with a Yeoman Angular installation, uses Bower for package management, and Grunt for building, minifying, testing, and deployment. Throw in some Jasmine and Karma for testing (unit testing and end-to-end testing), jQuery for advanced selectors, SASS for CSS, CouchDB for document based data storage, and then manage the versioning on GitHub using full issue management and ticket branching, and you've got a pretty over-engineered site.

What else? Ok, well instead of using spritesheets for the UI icons, this site uses an icon font. (50+ icons in a single 16Kb request? Yes please.)

You'll see the menu select indicator when you visit one of the other pages. Pure CSS, even the arrow.

The menus and page contents are loaded via REST api calls to the Couch database. The content utilizes Angular's $cacheFactory to prevent unecessary server calls.

This is a SPA (Single Page Application) style website, all page transitions and routing is handled with HTML5 style url/history modification. Fully bookmarkable at any location.

All animations are created as CSS transitions.

The special fonts are being loaded both from the Google Web Fonts CDN (menu and content fonts) and from this site (icon font) using Google's Web Font Loader library.

And... ah, yes. I designed it myself. Don't hate me for being beautiful. (Or not beautiful. Whatevs.)
